Calendar Booking Backend Service

A backend calendar booking system built with Node.js, Express, and Sequelize that allows users to schedule meetings while strictly preventing overlapping time slots.

This project is implemented as part of the Kraftshala Backend Assignment and follows a clean, modular architecture with proper validation and business rules.

 Features

Create and fetch users

Create, read, update, and delete meetings

Enforce no overlapping meeting times

Filter meetings by user and date range

Clean separation of concerns:

Routes

Controllers

Services

Models

Centralized error handling

SQL database using Sequelize ORM

 Tech Stack

Node.js (18+)

Express.js

Sequelize ORM

SQLite (used for local development)

dotenv

nodemon (dev dependency)
